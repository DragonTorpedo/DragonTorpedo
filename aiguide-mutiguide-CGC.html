The 3-Engine Playbook

Claude. Gemini. ChatGPT.

Three different engines built for three different jobs. Match the task — stop guessing.

Open the right one

Huge document or research dump → Gemini
Production code, refactors, long agent runs → Claude
Fast back-and-forth, steer it mid-task → ChatGPT
Reading video, audio, or images → Gemini
Writing that needs real nuance → Claude
Quick draft, brainstorm, simple chat → ChatGPT

Anthropic

Claude

The senior engineer. Give it the hard, messy problem.

Opus 4.7
Hardest jobs — big refactors, architecture, long agent runs.
Sonnet 4.6
Daily driver. Code review, writing, most tasks.
Haiku 4.5
Bulk speed. Classification, simple API calls.

Do this

  • Wrap inputs in <context> / <instructions> tags. Ask for <answer> back.
  • Say what to do, not what to avoid.
  • Hard problem? Just say "think thoroughly" — don't script the steps.
  • Give it a role and an output format in the same prompt.
Free local coding — route Claude Code through Ollama: claude --model qwen3-coder Set context to 64k+ first or long files get cut off. Recurring task? Save it once in .claude/skills/ — pulled in only when needed.

Google

Gemini

The researcher. Feed it everything — it won't lose the thread.

3.1 Pro
Hardest reasoning. Ambiguous tasks, judgment calls on images & video.
3.5 Flash
Daily driver. Near-Pro smarts, fastest, the default pick.
3.1 Flash-Lite
Bulk extraction, routing, high-volume classification.

Do this

  • Loading a big document? Put instructions at the bottom, after the content.
  • Skip the copy-paste — @Drive / @Gmail pull files straight in.
  • Treat video and audio like text. Just upload and ask.
  • Say "keep the output concise" or it over-explains.
The 1M-token window holds up under real load — load the whole repo, not snippets. That's the actual edge over the other two.

OpenAI

ChatGPT

The fast iterator. Steer it instead of restarting it.

GPT-5.5 Pro
Most thorough. Hardest, highest-stakes work.
GPT-5.5 Thinking
Daily driver. Coding, research, real documents.
GPT-5.5 Instant
Fast default. Quick chat, simple tasks.

Do this

  • Coding agent is Codex — message it mid-run to redirect, no restart needed.
  • Recurring job? Use Scheduled Tasks instead of re-prompting every time.
  • Write prompts like you'd talk — it rewards natural phrasing over rigid templates.
  • Lock in repeat context with a Project. Stop re-pasting the same brief.
Default model is Instant — switch to Thinking yourself for anything that actually matters.

Chain Them Together

The real unlock isn't picking one. It's handing off between them.

Design → Build

GeminiSketch the UI. Fast, cheap, good enough to throw away.
→
ClaudeWire up the logic. Debug it. Make it hold under real use.

Let Gemini throw shapes at the wall. Let Claude make them work.
